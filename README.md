## API Endpoints

- `GET /api/dashboard/overview` — Get municipal performance overview with key metrics
- `POST /api/kpis` — Create new KPI definition
- `PUT /api/kpis/:id` — Update KPI configuration or targets
- `GET /api/kpis/department/:dept` — Get KPIs for specific municipal department
- `POST /api/data/collect` — Submit performance data collection
- `GET /api/reports/generate/:type` — Generate specific report type (quarterly, annual, compliance)
- `GET /api/compliance/status` — Get current compliance status across all areas
- `POST /api/users` — Create new user account with role assignment
- `GET /api/notifications/pending` — Get pending notifications for current user
- `GET /api/audit/trail/:entity` — Get audit trail for specific entity or action
